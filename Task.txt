Creating  a REST API: TASKS AND OBJECTIVES-

GET /users - Give information of all the users. 
GET /user/id- Give the information of the user with ID= id 

POST /users - Create a user

PATCH /users/id- Update the infromation of the user with ID = id

DELETE /users/id - Delete user with ID=id 

***** Important Tips: *****
-For Dynamic entry of ID for users we need to use :id in route to get variable id values for users.
 Also called Dynamic Path Parameter.
